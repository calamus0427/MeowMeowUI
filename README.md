# [MeowMeowUI](https://calamus.wiki/MeowMeowUI/)

> MeowMeowUI

基于Vue2.0+的UI组件库

## 文档
- [中文文档](https://calamus.wiki/MeowMeowUI/)

## Start
### 下载
```bash
npm install meowmeowui
```
### 使用
```js
//main.js
import MeowUi from '../packages/index'
Vue.use(MeowUi)
```
